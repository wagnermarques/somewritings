** Conhecimentos Prévios
   Vc deve saber utilizar o android studio
   Utilizar comandos do gradle e do android sdk   
   Os comandos do gradle que vc precisa saber são:
   #+BEGIN_EXAMPLE
     diretoriodoprojeto>gradlew installDebug
     adb shell am start -n pacotedoprojeto\.NomeDaActivity
   #+END_EXAMPLE
   Você pode consultar o nome do pacote do projeto no androidManifest.xml e
   também o nome da activity que vc vai querer abrir.

** Atividade Prática   
*** Fazendo as modificações no seu projeto pra ele ser um jetpack
   Só precisa fazer isso se vc tiver usando o android studio 3.4, se
   tiver usando o 3.5 não precisa.
   
   Dá pra simplificar esse processo de transformar o seu projeto em um
   projeto jetpack apenas 
   1) copiando o conteúdo dos meus dois arquivos build.gradle do meu projeto
   2) copiando o conteúdo do meu gradle.properties
