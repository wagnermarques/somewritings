* Apresentacao deste texto pra Aula de Hoje
  A gente vai fazer de novo o listview.
  Na verdade vários de vcs fizeram na aula passada, mas a maioria não.
  Na aula passada eu deixei só essas tres imagens abaixo, que vcs
  devem ter aí com vcs também...
  
  Estou colocando elas abaixo novamente pra gente conversar sobre elas
  mas também, neste texto, tem um tutorial utilizando outras imagem
  completamente diferentes que pode facilitar pro pessoal que teve
  dificuldade com essas imagens.
 
  A diferença entre o trabalho dessas três imagens e do novo tutorial
  que estou trazendo pra vcs nessa aula é que na aula passada vcs
  tinham que criar um botão que abria a listview.

  No tutorial novo, neste documento, a activity principal já é a list
  própria listview.


  Última palavrinha pra vcs começaram a aula:
  Criar uma listview básico com um array de strings é fácil é só
  seguir esses três passos:
  + Criar uma activity
    Claro porque a listview tem que aparecer em uma activity
    Detalhe: Na figura CodigoPraAbrirA_ListViewActivity.png o que eu
    estou mostrando é minha activity principal criada sem xml, usando
    anko. Não é a active da listview. Essa activity mostra o botão com
    a linha startActivity<ListViewActivity>() que é a que abre o a
    activity da listvew.
    O que importa é que vc vai precisar de uma activity pra listview
    propriamente dita. Neste caso vc só precisava olhar a imagem
    codigo_da_ListViewActivity.png que é a segunda apresentada abaixo.

  + Inserir o widget da list view no xml da sua activity
    Isso está na imagem ListViewActivity-LayoutXML.png
    
  + Aí, no código da Activity da ListView vc inicia ela
    Isso tá na imagem codigo_da_ListViewActivity.png
    O código simplesmente cria uma ArrayList de string nas linhas 24
    a 29.
    A linha 23 obtem o widget da list view e a linha 30 insere os
    dados na listview.
    A linha 30 tem vários detalhes pra entender mas vc pode colocar o
    que vc quizer na sua array e usar ela como modelo pra mostrar uma
    listvew simples.
