* Tema
  Migrar para jetpack
* Objetivo
  Mostrar a diferença entre uma aplicação jetpack e nao jetpack
* Justificativa
  Saber transformar uma aplicação nao jetpack pra jetpack é importante
  porque jetpack é a nova tecnologia que a google está utilizando.
  Se vc tiver com o android studio 3.5 os seus projetos já são criados
  usando jetpack, mas no nosso caso, que é o 3.4, não usa jetpack.

  Qdo a gente for estudar google maps vamos precisar de jetpack, por
  isso precisamos dessa aula.
  Além disso, essa aula vai ser importante pra vcs conhecerem a
  estrutura de um projeto android.

* Duração
  2Blocos
