* Procedimento didatico
  
  Aula expositiva e prática em laboratório  
  
  Nós vamos utilizar o projeto [[https://github.com/wagnermarques/somewritings/tree/rascunho/android/projetos/LoginAppJava][LoginAppJava]] para estudá-lo e depois
  fazer o nosso próprio projeto.
  
  O projeto é uma activity que recebe usuário e senha e vê se confere
  com os dados internos. Se tem o usuário e senha informado ele
  retorna login com sucesso utilizando toast caso contrário, falha no
  login.

  Além disso ele mostra vários toast que mostram como pegamos os dados
  dos widgets da activity.

   Lembrando que o trecho abaixo serve pra fazer com que sua activity
   abra quando o seu usuário inicia a aplicação. 
   Tecnicamente falando, sua activity consegue responder a intent que 

    #+BEGIN_SRC xml
    <intent-filter>
                <action android:name="android.intent.action.MAIN" />

                <category android:name="android.intent.category.LAUNCHER" />
    </intent-filter>
   #+END_SRC
