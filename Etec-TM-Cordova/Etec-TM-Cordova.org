* Template :noexport:
Template é um modelo que de site que vc pode aplicar no seu projeto
** usando um template
Entra no site do npm e pesquisa por "cordova:template"
Encontrou o template? escolha uma das pções abaixo pra criar o projeto
usando o template. [fn:cordovatemplates]
#+BEGIN_SRC sh
$ cordova create hello com.example.hello HelloWorld --template <npm-package-name>
$ cordova create hello com.example.hello HelloWorld --template <git-remote-url>
$ cordova create hello com.example.hello HelloWorld --template <path-to-template>
#+END_SRC
