#+Title: windows10

* Otimizações
** reduzir os serviços na inicialização do sistema
  msconfig -> aba Serviços -> checar "Ocultar todos os serviços
  Microsoft" -> clicar em "Desativar Tudo"

  gerenciador de tarefas -> aba "Inicializar" -> aí vc desabilita os
  serviços não microsoft e outros que vc não precisar

** apagar arquivos temporários
   win+r -> digite "%temp%" -> apague os arquivos

** desabilitar a criação automática de pontos de restauração do sistema  
   desabilite criação automática de pontos de recuperação (aí vc tem
   que criar pontos manualmente)
  
   Botão direito no "este computador" -> Aba "Proteção do Sistema" ->
   btn 'Configurar' -> marque a opc "desativar a proteção do sistema" e
   "em uso do espaço em disco" deixe em 0%


* resolver problemas de atualização
** usar a ferramenta dism[fn:msdism]
   com um prompt executando como administrador  
   Dism /Online /Cleanup-Image /ScanHealth
   DISM.exe /Online /Cleanup-image /Restorehealth

* Manutenção do windows update
** Método automático pela resoluçao de problemas do windows
** Método de limpeza manual do cache do windows update
   shift + reiniciar (pra reiniciar no modo de segurança)
   Pare o serviço do windows update: net stop wuauserv 
   Apague o conteúde de C:/Windows/SoftwareDistribution
   Reinicie o serviço: net stop  wuauserv
