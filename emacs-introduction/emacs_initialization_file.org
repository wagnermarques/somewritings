* A inicialização do emacs

  Quando vc liga o emacs, durante a inicialização o emacs vai procurar
  e carregar os seguintes arquivos. Caso sejam encontrados o emacs vai
  executar eles. A procura é feita na seguinte ordem (testar essa ordem):

  ~/.emacs
  ~/.emacs.el
  ~/.emacs.d/init.el

  Meu diretório ~./emacs tem os seguintes arquivos...
  
   #+NAME:

* fornecendo um arquivo de inicializacao para o emacs
  Vou optar por criar um arquivo init.el no meu diretório ~/.emacs.d
  
  
   #+NAME: criando init.el                   
   #+BEGIN_SRC shell :session s1 :results output :exports both
      #starts code
      touch ~/.emacs.d/init.el
      echo '(message "Arquivo init.el carregado com sucesso!")' >> ~/.emacs.d/init.el
      cat ~/.emacs.d/init.el
   #+END_SRC

   #+RESULTS: criando init.el
   : 
   : [wagner@nsi_pc_149_3 emacs-introduction]$ [wagner@nsi_pc_149_3 emacs-introduction]$ (message "Arquivo init.el carregado com sucesso!")
   : (message "Arquivo init.el carregado com sucesso!")

   Agora quando abrimos o nosso emacs, podemos ver que o emacs agora
   carregou e rodou esse arquivo durante o processo de inicialização.
