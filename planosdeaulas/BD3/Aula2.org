* Avaliação
  

* Gerenciador Visual de Banco de dados
  Podemos interagir com o banco de dados sqlserver de varias
  maneiras. A maneira mais comum, que costumavamos usar nos
  laboratorios era conectar no banco usando o ssms sqlserver 
  managment studio.[fn:ssmsSite]
  
  Se preferir vc pode instalar ele na sua maquina. Na minha vou usar o
  o squirrel que fornece meios para conectar com outros banco de dados
  tambem. [fn:squirrelSite]

  Podemos usar a linha de comando tambem, usando o sqlcmd. A gente nao
  viu isso no lab mas he uma opcao.


* Conectando com o servidor sqlserver
  Pra fazer os exercicios vc pode instalar o sqlserver na sua maquina,
  ou usar o sqlserver que eu instalei na azure pra gente usar. Tanto
  faz.

  Pra conectar no sqlserver da azure utilize os seguintes parametros:

  url:
  jdbc:jtds:sqlserver://eteczl-sqlserver.database.windows.net:1433/master

  usuarios: administrador
 
  senha: o prof vai passar pra vcs

   
  Usando o squirrel ficaria mais ou menos como na imagem abaixo,
  lembrando que o squirrel usa java pra fazer a conexao e por isso he
  necessario baixar o driver jdbc[fn:sqlserverJdbcDriver] pra tudo funcionar.
  
  O professor vai explicar na live como faz essa conexao
