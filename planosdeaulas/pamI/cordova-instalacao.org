* Instalacao do cordova

  Usar o npm do nodejs conforme abaixo

  
#+NAME:npm install cordova
#+BEGIN_SRC shell :session s1 :results output :exports both

#+END_SRC


* Verificar se o cordova esta funcionando
  
** Criar um projeto do cordova
  
#+NAME: cordova create Proj1DeNovo
#+BEGIN_SRC shell :session s1 :results output :exports code :eval no
  cordova create Proj1DeNovo
  cd Proj1DeNovo
#+END_SRC

   Da um dir (windows) ou um ls(linux) pra ver os arquivos do seu novo projeto.

** Estrutura do Projeto

*** config.xml
    Arq xml de  configuracoa do projeto
*** platforms
    Vamos trabalhar so com a plataforma android
*** plugins
    Local dos  plugins do seu projeto, como camera etc
*** www
    Aqui fica os arquivos html, javascript e css do seu projeto

** Rodando o seu projeto
*** No Browser    
#+NAME:cordova platform add browser
#+BEGIN_SRC shell :session s1 :results output :exports code
  cd Proj1DeNovo #entra no diretorio do seu projeto
  cordova platform add browser
  coreova run browser
#+END_SRC
*** No emulador do android
    Depois a gente ve isso... 
*** No Dispositivo
#+NAME:cordova platform add android
#+BEGIN_SRC shell :session s1 :results output :exports code
  cd Proj1DeNovo #entra no diretorio do seu projeto
  cordova platform add android
  adb devices #tem que por o adb no path do OS
  coreova run android
#+END_SRC
